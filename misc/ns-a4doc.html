a4doc Namespace

Note: resources are being published as they (continually) emerge.

a4doc namespace serves a purpose of annotating specific notable semantic properties accompanied with (optional) textual details or just standalone hints connected with annotated entity.

Common attributes

a4doc:hint

standalone side notes for the item

a4doc:deprecation-hint

details about item deprecation (when used, even if with an empty string, marks the current item as deprecated)

a4doc:danger-hint

explaining what is dangerous about touching this item (when used, even if with an empty string, marks the current item as dangerous/experimental)

a4doc:discretion-hint

explaining why extra caution about touching this item (when used, even if with an empty string, marks the current item as requiring discretion)
